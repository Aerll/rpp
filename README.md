# About
**rules++** is nothing more than **rules** on steroids. It's doing the exact same thing, but better. The language was implemented in C++.

## Getting started
All you need are 3 files:
  - **rules++.exe** - an executable, which parses **rules++** code and translates it into **rules**
  - **base.r** - an implementation of all basic functions
  - **base.p** - a collection of useful presets
  
To learn mote about the language, follow a tutorial on [wiki](https://github.com/Aerll/rpp/wiki).

## Utilities for notepad++
Utilities include syntax highlighting and auto completion. Preferable theme can be found [here](https://github.com/cydh/VS2015-Dark-Npp).<br/>
Don't forget to check _Enable global background colour_ in _Settings -> Style Configurator..._
